A smart, AI-driven solution built to help authorities monitor, analyze, and predict water-logging situations across Delhi. The system focuses on proactive decision-making using data analytics and Bayesian Ensemble Machine Learning to identify high-risk wards before emergencies escalate.

## Key Features
- Role-based dashboards (Admin, Supervisor, Analyst, Field Officer)
- Centralized water-logging complaint management
- Ward-wise analytics and visual insights
- Bayesian Ensemble ML models for risk prediction
- Explainable, confidence-aware AI outputs

## Technology Stack
- HTML, CSS, JavaScript  
- Chart.js for visualizations  
- Python (scikit-learn, PyMC3) for Machine Learning  
- CSV-based datasets  
- Git & GitHub
